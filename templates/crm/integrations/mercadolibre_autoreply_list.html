Como funciona

Cuando llega una pregunta, el sistema busca coincidencias con las palabras clave de tus plantillas. Si encuentra una coincidencia, responde automaticamente. Las plantillas con mayor prioridad se evaluan primero. Si ninguna plantilla coincide, se usan reglas basicas de fallback (stock, envio, precio, etc.).
